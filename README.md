# Interactive 3D Solar System

A cinematic, interactive 3D Solar System experience built with Three.js and Vite.

## Features
- **Immersive 3D Environment**: Realistic lighting, starfield, and smooth camera controls.
- **Interactive Planets**: Click on any planet to focus and view details.
- **Smooth Transitions**: Cinematic camera movements between planets.
- **Educational Data**: Real data about planets (scaled for visual clarity).
- **Asteroid Belt**: Visual representation of the asteroid belt.

## Tech Stack
- **Three.js**: 3D Rendering
- **Vite**: Development server and bundler
- **Vanilla JavaScript**: ES Modules
- **HTML5 & CSS3**: UI and styling

## How to Run

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```

3.  **Open in Browser**:
    Visit the URL shown in the terminal (usually `http://localhost:5173`).

## Controls
- **Left Click + Drag**: Rotate Camera
- **Scroll**: Zoom In/Out
- **Click Planet**: Focus and View Info
- **Reset Button**: Return to Solar System View
